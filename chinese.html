欢迎使用中文提示词库

涵盖教学规划、评估、学生支持等多个类别，助力华文教育工作

📚 通用兼容提示词

这些提示词适用于所有中文大语言模型，包括 DeepSeek、Qwen、Kimi、Doubao、MiniMax 等。

📝 课程规划

1. 课程单元设计
你是一位经验丰富的华小/独中教师。请帮我设计一个[科目名称]的教学单元,主题是[主题],适用于[年级]学生。 要求: - 符合马来西亚华小KSSR/KSSM课程标准(或独中统考课程标准) - 教学时长:[X]节课 - 包含学习目标、教学活动、评估方式 - 考虑学生的中文程度和文化背景 - 提供差异化教学建议 请以表格形式呈现教学流程。
2. 每日教案生成
请为我准备一份[科目]教案,课题是[课题名称],对象是[年级]的华小/独中学生。 教案需包含: 1. 学习目标(认知、技能、情意) 2. 教学准备(教具、材料) 3. 导入活动(5-10分钟) 4. 发展活动(20-30分钟) 5. 巩固活动(10-15分钟) 6. 总结与评估 7. 作业安排 请确保教学活动有趣且适合华文教学环境。
3. 跨学科整合课程
请帮我设计一个跨学科整合课程,结合[科目1]和[科目2],主题为[主题],适合[年级]华小/独中学生。 设计要点: - 体现两个科目的核心概念 - 设计真实情境的学习任务 - 融入马来西亚本土文化元素 - 提供协作学习机会 - 包含多元评估方式

📊 评估与考试

4. 形成性评估设计
请为[科目]的[课题]设计形成性评估活动,对象是[年级]学生。 要求: - 提供3-5种不同的评估方式 - 包含评估标准或检核表 - 适合课堂即时反馈 - 能诊断学生的学习困难 - 提供改进教学的建议
5. 课堂活动创意
请为[科目]课题[课题名称]设计5个有趣的课堂活动,适合[年级]华小/独中学生。 活动要求: - 符合学习目标 - 适合[班级人数]人的班级 - 时间控制在[X]分钟内 - 需要的材料容易取得 - 能激发学生参与和互动 - 适合华文教学环境

🎯 学生支持

6. 差异化教学策略
我班上有不同程度的学生。请为[科目]课题[课题名称]提供差异化教学策略。 学生分组: - 基础组:需要额外支持 - 中等组:符合年级水平 - 进阶组:需要挑战 请分别提供适合各组的: 1. 学习材料调整 2. 教学活动设计 3. 评估方式 4. 延伸学习建议
7. 学习困难诊断与干预
我有一位[年级]学生在[科目]方面遇到困难,具体表现为:[描述问题]。 请帮我: 1. 分析可能的原因 2. 设计诊断评估活动 3. 提供针对性的教学干预策略 4. 建议家长可以如何配合 5. 制定进步追踪计划

🏫 课堂管理

8. 课堂常规建立
请帮我为[年级]班级设计一套有效的课堂常规和规则。 要求: - 符合华小/独中的校园文化 - 语言正面且易于理解 - 包含3-5条核心规则 - 提供规则教导活动 - 建议奖励与后果系统 - 适合中文环境呈现

💡 专业发展

9. 教学反思
请帮我进行教学反思。今天我教了[科目]的[课题],对象是[年级]。 课堂情况: - 进行顺利的部分:[描述] - 遇到的挑战:[描述] - 学生反应:[描述] 请帮我: 1. 分析课堂的优点和不足 2. 提供改进建议 3. 建议下次教学的调整方向

🔍 DeepSeek 专用提示词 推理能力强

DeepSeek 模型特点:强大的推理能力、数理逻辑、编程支持、深度分析

1. 数学解题思路分析
你是一位数学教学专家。请用深度推理的方式分析这道数学题,并提供详细的教学指导。 题目:[数学题目] 年级:[年级] 学生常见错误:[如有] 请提供: 1. 题目的数学概念分析 2. 多种解题方法(至少3种) 3. 每种方法的思维步骤 4. 学生可能的思维障碍 5. 教学引导策略 6. 相关的练习题(附答案) 请用结构化的方式呈现,便于教学使用。
2. 科学概念深度解析
请深度解析[科学概念],适合[年级]华小/独中学生理解。 要求: 1. 概念的科学原理(精确但易懂) 2. 与日常生活的联系(马来西亚情境) 3. 常见的迷思概念及纠正方法 4. 实验或演示活动设计 5. 进阶思考问题 6. 跨学科连接 请运用你的推理能力,建立清晰的知识架构。
3. 编程教学活动设计
请为[年级]学生设计一个编程教学活动,主题是[主题]。 要求: 1. 使用适合年龄的编程工具(Scratch/Python等) 2. 结合[科目]的学习内容 3. 提供完整的代码示例和注释 4. 设计阶段性任务(从简单到复杂) 5. 常见错误和调试建议 6. 延伸挑战任务 请提供清晰的教学步骤和代码。
4. 逻辑思维训练题库
请为[年级]学生创建一套逻辑思维训练题,共[X]题,主题是[主题]。 要求: 1. 题目难度递增 2. 涵盖不同类型的逻辑推理 3. 每题提供详细解题思路 4. 分析题目考查的思维能力 5. 提供变式练习 6. 适合华小/独中学生的认知水平

🌐 Qwen 通义千问专用提示词 多模态

Qwen 模型特点:多语言能力、多模态理解、文化适应性强、创意与实用并重

1. 双语教学材料开发
请为[科目]的[课题]开发双语教学材料,适合[年级]马来西亚华小学生。 要求: 1. 主要语言:中文 2. 辅助语言:马来语/英语关键词 3. 设计原则:中文为主,适当融入其他语言 4. 包含词汇表(中英马对照) 5. 文化敏感度考量 6. 适合多语言环境的教学活动 请确保语言切换自然,符合马来西亚教育情境。
2. 多媒体教学内容规划
请为[课题]规划一个多媒体教学内容,适合[年级]学生。 内容包含: 1. 教学影片脚本(5-8分钟) 2. 配套的视觉元素描述(图片、图表、动画) 3. 互动元素设计(提问、小测验) 4. 字幕和旁白文本 5. 课后延伸活动 请描述每个多媒体元素的教学功能和呈现方式。
3. 文化融合课程设计
请设计一个融合中华文化与马来西亚本土文化的[科目]教学单元,主题是[主题],适合[年级]。 要求: 1. 平衡两种文化元素 2. 促进文化理解和欣赏 3. 包含实践活动(如节日、习俗、艺术) 4. 设计跨文化比较任务 5. 培养多元文化视野 6. 适合华小/独中多元环境 请提供丰富的文化背景资料和教学建议。
4. 图文理解教学活动
请为[年级]学生设计一个图文理解教学活动,主题是[主题]。 活动包含: 1. 选择/设计合适的图片或图表 2. 设计观察和分析问题(从表层到深层) 3. 图文对应练习 4. 创意表达任务(学生自制图文作品) 5. 评估标准 请详细描述如何引导学生发展多模态理解能力。

🎨 MiniMax 专用提示词 创意生成

MiniMax 模型特点:创意生成、故事叙述、艺术设计、想象力激发

1. 创意写作教学
请设计一个创意写作教学活动,主题是[主题],适合[年级]华小/独中学生。 活动包含: 1. 创意启发活动(暖身游戏) 2. 写作提示和框架 3. 范文示例(你创作的优秀范文) 4. 修改和润色指导 5. 创意展示方式 6. 评估标准(着重创意和表达) 请发挥创意,让写作变得有趣。
2. 故事化学习内容
请将[科目]的[概念/课题]转化为一个有趣的故事,适合[年级]学生。 故事要求: 1. 主角设定(学生能认同的角色) 2. 情节发展(融入学习内容) 3. 问题和挑战(对应知识点) 4. 解决方案(体现学习目标) 5. 寓意和启发 故事长度:约[字数]字 请确保故事引人入胜,同时准确传达学习内容。
3. 艺术整合项目设计
请设计一个艺术整合项目,结合[科目]和艺术创作,主题是[主题],适合[年级]。 项目包含: 1. 学习目标(学科+艺术) 2. 创作媒介建议(绘画/手工/表演/数字艺术) 3. 创作步骤指导 4. 学科知识的创意呈现方式 5. 作品展示和分享活动 6. 评估标准(学科理解+艺术表现) 请提供激发创意的具体建议。
4. 游戏化学习设计
请将[课题]设计成一个教育游戏,适合[年级]学生在课堂使用。 游戏设计: 1. 游戏类型(角色扮演/竞赛/闯关/协作等) 2. 游戏规则(简单易懂) 3. 学习目标融入方式 4. 游戏流程(15-30分钟) 5. 所需材料/道具 6. 计分/奖励系统 7. 教师引导要点 请确保游戏既有趣又有学习价值。

📝 考试相关工具

1. 试卷命题助手
请为[科目]出一份[类型]试卷,适合[年级]华小/独中学生。 试卷要求: - 考试时长:[X]分钟 - 总分:[X]分 - 题型分配:[客观题X分,主观题X分] - 难度比例:基础[%]、中等[%]、挑战[%] - 课题范围:[列出课题] - 符合KSSR/KSSM或独中统考格式 请提供: 1. 完整试卷(包含所有题目) 2. 标准答案 3. 评分标准 4. 各题考查的知识点和能力
2. 答案详解生成器
这是一份[科目]试卷的题目。请为每道题提供详细的答案解析,适合教师教学使用。 题目:[贴上题目] 请提供: 1. 正确答案 2. 详细解题步骤 3. 考查的知识点 4. 学生常见错误 5. 教学建议 6. 相似题型 请确保解析清晰易懂,便于教师讲解。
3. 评分标准/量规制定
请为以下[科目]的[题型/任务]制定详细的评分标准(Rubric)。 任务描述:[描述任务] 年级:[年级] 总分:[X]分 量规要求: 1. 3-5个评分维度 2. 每个维度4-5个等级描述 3. 清晰的评分指标 4. 适合华小/独中评估标准 5. 提供评分示例 请以表格形式呈现量规。
4. 错题分析与补救教学
以下是学生在[课题]测验中的常见错误: 错题统计: - 题目X:错误率[%],常见错误:[描述] - 题目Y:错误率[%],常见错误:[描述] 请帮我: 1. 分析错误的根本原因 2. 诊断学生的概念理解问题 3. 设计补救教学活动(20-30分钟) 4. 提供针对性练习题 5. 建议后续评估方式 请提供实用的教学策略。
5. 参考答案方案生成
请为这道[科目]主观题提供完整的参考答案方案,适合[年级]水平。 题目:[贴上题目] 分数:[X]分 请提供: 1. 完整的标准答案 2. 答题要点(分点列出) 3. 每个要点的分数分配 4. 关键词/关键概念 5. 不同等级的答案示例(优秀/良好/及格) 6. 评分注意事项 请确保答案方案严谨且便于教师评分。
6. 试题库建立
请为[科目]的[课题]建立一个试题库,适合[年级]。 要求: - 题型:[选择题/填空题/简答题/应用题等] - 题量:每种题型[X]题 - 难度分布:基础/中等/挑战 - 涵盖所有学习目标 - 提供完整答案 - 标注课题标签和难度等级 请按题型分类组织题目。

💡 使用建议

选择合适的模型:

  • DeepSeek: 适合数理科、逻辑推理、深度分析、编程教学
  • Qwen: 适合语文、双语教学、文化课题、多媒体整合
  • MiniMax: 适合创意写作、艺术课、故事教学、游戏化学习
  • 通用提示词: 适合所有日常教学需求

提示词使用技巧:

  1. 填写方括号 [ ] 内的信息，越具体越好
  2. 提及华小/独中、KSSR/KSSM/统考等具体情境
  3. 根据实际需求调整提示词的要求部分
  4. 从简单要求开始，逐步添加细节
  5. 如需要 technical terms 可适当使用英语
